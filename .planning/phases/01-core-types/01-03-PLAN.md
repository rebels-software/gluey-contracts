Create src/Gluey.Contract/ParseResult.cs:

1. public readonly struct ParseResult : IDisposable
2. Fields:
   - OffsetTable _offsetTable
   - ErrorCollector _errorCollector
   - Dictionary<string, int> _nameToOrdinal (or FrozenDictionary<string, int>) -- this is the schema's name-to-ordinal mapping, passed in at construction. Per research Open Question 2: the mapping belongs to the schema but ParseResult needs it for the string indexer.

3. Internal constructor: ParseResult(OffsetTable offsetTable, ErrorCollector errorCollector, Dictionary<string, int> nameToOrdinal)

4. Public properties:
   - bool IsValid => !_errorCollector.HasErrors (if _errorCollector is default/null-backed, return true)
   - ErrorCollector Errors => _errorCollector (expose the full collector so users can iterate errors)

   NOTE on Errors: Per user decision, "Errors collection always accessible via ParseResult.Errors -- empty on success, populated on failure (uniform API)". Expose as a readonly view. The simplest approach: expose ErrorCollector directly since it is a readonly struct. Users can foreach, index, and check Count.

5. Public indexers:
   - ParsedProperty this[int ordinal] => _offsetTable[ordinal] (delegates to OffsetTable)
   - ParsedProperty this[string name] => looks up ordinal in _nameToOrdinal, then delegates to _offsetTable[ordinal]. If name not found, returns ParsedProperty.Empty.

6. Public method: Enumerator GetEnumerator()
   - Custom struct enumerator that iterates _offsetTable entries [0..Count) returning ParsedProperty values
   - Only yields entries where HasValue == true (skip empty slots)

7. Dispose():
   - _offsetTable.Dispose()
   - _errorCollector.Dispose()

8. Handle default/uninitialized state gracefully: if _offsetTable or _errorCollector are default structs, all operations should return safe defaults (no NullReferenceException).

9. XML doc comments on all public members.

Per user decisions:
- Both string and ordinal indexers resolve to the same offset table
- Missing property returns empty ParsedProperty (HasValue = false, no exceptions)
- Supports foreach via GetEnumerator()
