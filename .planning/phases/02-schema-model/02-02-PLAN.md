Create `src/Gluey.Contract.Json/JsonSchemaLoader.cs` as an `internal static class` with:

**Entry point:**
```csharp
internal static SchemaNode? Load(ReadOnlySpan<byte> utf8Json)
```
Creates a `Utf8JsonReader` and delegates to `ReadSchema(ref reader, "")`.

**Core recursive method:**
```csharp
private static SchemaNode? ReadSchema(ref Utf8JsonReader reader, string currentPath)
```
- First checks token type: if `True`/`False` return `SchemaNode.True`/`SchemaNode.False`
- If not `StartObject`, return null
- Reads property names in a while loop, matching each against known keywords using `reader.ValueTextEquals("keyword"u8)` for zero-alloc matching
- For each keyword category, delegates to typed reader helpers
- Unknown keywords: `reader.Skip()` to silently skip the value
- At end of object, constructs SchemaNode with all collected values

**Keyword reader helpers (private static, all take `ref Utf8JsonReader`):**
- `ReadType` -- handles string or array of strings, returns `SchemaType?`
- `ReadSchemaOrBoolean` -- handles sub-schema (object) or boolean schema (true/false), used for additionalProperties, items, contains, not, if, then, else, propertyNames
- `ReadSchemaArray` -- reads `[schema, ...]`, used for allOf, anyOf, oneOf, prefixItems
- `ReadSchemaMap` -- reads `{"key": schema, ...}`, used for properties, patternProperties, dependentSchemas, $defs. Builds child paths using `SchemaNode.BuildChildPath`
- `ReadStringArray` -- reads `["a", "b"]`, used for required
- `ReadDependentRequired` -- reads `{"key": ["a", "b"], ...}`
- `ReadEnum` -- reads array of any JSON values, serializing each to `byte[]` via a small buffer + `Utf8JsonWriter`
- `ReadConst` -- reads any JSON value, serializing to `byte[]` via `Utf8JsonWriter`
- `ReadDecimal` -- `reader.GetDecimal()`
- `ReadInt` -- `reader.GetInt32()`
- `ReadString` -- `reader.GetString()`
- `ReadBool` -- `reader.GetBoolean()`

**Critical implementation notes:**
- ALWAYS pass Utf8JsonReader by `ref` (it's a mutable ref struct; value copies break state)
- For enum/const serialization to byte[]: use `ArrayBufferWriter<byte>` + `Utf8JsonWriter` to serialize the current value, then `.WrittenSpan.ToArray()`
- For ReadSchema calls on sub-schemas: reader.Read() to advance to value token, then call ReadSchema
- Property name matching order: start with most common keywords (type, properties, required, items) for early exit
- Path building: use `SchemaNode.BuildChildPath(currentPath, propertyName)` only for `properties` and `patternProperties` children (they are named properties)
- For composition keywords (allOf etc.), child paths use array index syntax but these don't get ordinals -- just use the parent path for now

`internal static class SchemaIndexer` with:
```csharp
internal static (Dictionary<string, int> nameToOrdinal, int propertyCount) AssignOrdinals(SchemaNode root)
```
- Walks the SchemaNode tree depth-first
- Only named object properties (from `Properties` dictionaries) get ordinals
- Uses the SchemaNode's precomputed `Path` as the dictionary key
- Recurses into: Properties children, Items, PrefixItems, AllOf/AnyOf/OneOf children, Not, If/Then/Else, Contains, AdditionalProperties, PatternProperties, DependentSchemas, $defs
- Array items themselves do NOT get ordinals
- Returns the mapping and the total count
