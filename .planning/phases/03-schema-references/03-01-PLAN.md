- Internal storage: `Dictionary<string, SchemaNode>` -- no concurrency needed (schema loading is not concurrent per design)
- Public API surface:
  - `void Add(string uri, SchemaNode root)` -- stores root node keyed by URI. - URI normalization: trim trailing slashes, use ordinal comparison. Keep it simple per Claude's discretion.
- The class itself is `public` so it can be passed as a parameter to JsonContractSchema.TryLoad/Load (public API), but its mutation methods are internal since they accept/return internal types.
